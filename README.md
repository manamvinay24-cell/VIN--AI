# VINAI AI Career Assistant

VINAI is an AI-powered career assistant web app for students.

## Features
- Login page
- Dashboard
- Resume Builder
- Saved Resumes
- Cover Letter Helper
- Why Hire Me Generator
- Internship Tracker
- AI Chat Assistant
- Skills Roadmap

## Tech Stack
- HTML
- CSS
- JavaScript
- Netlify Functions
- Gemini API

## Live Demo
https://vinai-resume-builder.netlify.app

## About
I built this project to practice web development and AI integration.  
This app helps students create resumes, track internships, and ask career or coding questions using AI.
